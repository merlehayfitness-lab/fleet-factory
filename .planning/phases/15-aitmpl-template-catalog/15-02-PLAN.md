    **apps/web/_actions/aitmpl-actions.ts** (NEW):
    "use server" file with Server Actions for AITMPL catalog operations.

    Imports:
    ```typescript
    "use server";

    import { createServerClient } from "@/_lib/supabase/server";
    import { redirect } from "next/navigation";
    import { revalidatePath } from "next/cache";
    import {
      searchComponents,
      getComponentDetail,
      importFromAitmpl,
      getCatalogStats,
    } from "@fleet-factory/core/server";
    import type {
      CatalogSearchResult,
      AitmplComponentType,
      AitmplImportResult,
    } from "@fleet-factory/core";
    ```

    Export `async function searchAitmplAction(query, options?)`:
    - Params: `query: string`, `options?: { type?: AitmplComponentType; department?: string; limit?: number; sort?: "downloads" | "name" | "newest" }`
    - Returns: `Promise<{ results: CatalogSearchResult[] } | { error: string }>`
    - Auth check: get user, if !user return `{ error: "Not authenticated" }`
    - try/catch: call `searchComponents(query, options)`, return `{ results }`
    - On error: return `{ error: err.message }`

    Export `async function getAitmplDetailAction(path, type)`:
    - Params: `path: string`, `type: AitmplComponentType`
    - Returns: `Promise<{ component: { name: string; path: string; category: string; type: string; content: string; description: string; downloads: number } } | { error: string }>`
    - Auth check
    - try/catch: call `getComponentDetail(path, type)`
    - If null, return `{ error: "Component not found" }`
    - Return only needed fields (don't expose full raw object with security etc to client): `{ component: { name, path, category, type, content, description, downloads } }`

    Export `async function importAitmplAction(options)`:
    - Params: `options: { businessId: string; componentPath: string; componentType: AitmplComponentType; targetAgentId?: string; targetDepartmentId?: string }`
    - Returns: `Promise<AitmplImportResult>`
    - Auth check: get user, if !user `redirect("/sign-in")`
    - try/catch: call `importFromAitmpl(supabase, options)`
    - On success: `revalidatePath(\`/businesses/${options.businessId}\`)`
    - Return the result

    Export `async function getAitmplStatsAction()`:
    - Returns: `Promise<{ stats: Record<string, number> } | { error: string }>`
    - Auth check
    - try/catch: call `getCatalogStats()`, return `{ stats }`
  </action>
