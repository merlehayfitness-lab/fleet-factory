    **infra/vps/terminal-bridge.ts** (NEW):
    ssh2 WebSocket-to-SSH bridge for the embedded terminal.

    ```typescript
    import { Client } from "ssh2";
    import type { WebSocket } from "ws";
    ```

    Export `function bridgeTerminal(ws: WebSocket, businessSlug: string): void`:

    1. Create ssh2 Client: `const conn = new Client();`

    2. Track connection state: `let sshReady = false;`

    3. On `conn.on("ready")`:
       - Set `sshReady = true`
       - Call `conn.shell({ term: "xterm-256color", cols: 80, rows: 24 }, (err, stream) => { ... })`
       - If err: send error message to ws, close ws, return

       Inside the shell callback:
       - **SSH -> WebSocket**: `stream.on("data", (data: Buffer) => { if (ws.readyState === 1) ws.send(data); })`
       - **WebSocket -> SSH**: `ws.on("message", (data: Buffer | string) => { ... })`
         - Try parse as JSON for resize commands:
           ```typescript
           try {
             const msg = JSON.parse(data.toString());
             if (msg.type === "resize" && msg.cols && msg.rows) {
               stream.setWindow(msg.rows, msg.cols, 0, 0);
               return;
             }
           } catch { /* not JSON, raw terminal input */ }
           stream.write(data);
           ```
       - **Cleanup**: `stream.on("close", () => { ws.close(); conn.end(); });`
       - **Cleanup**: `ws.on("close", () => { stream.close(); conn.end(); });`

       - **Initial directory**: After small delay (100ms), send:
         ```typescript
         stream.write(`cd /data/tenants/${businessSlug} 2>/dev/null && clear\n`);
         ```

    4. On `conn.on("error")`:
       - Send error message as text to ws: `ws.send(\`\r\nSSH connection error: ${err.message}\r\n\`)`
       - Close ws

    5. On `ws.on("close")`:
       - If ssh not ready yet, end the connection attempt: `conn.end()`

    6. Connect using env vars:
       ```typescript
       const connectConfig: any = {
         host: "127.0.0.1",
         port: 22,
         username: process.env.SSH_USER || "root",
       };

       const keyPath = process.env.SSH_PRIVATE_KEY_PATH;
       if (keyPath) {
         connectConfig.privateKey = require("node:fs").readFileSync(keyPath);
       } else {
         connectConfig.password = process.env.SSH_PASSWORD || "";
       }

       conn.connect(connectConfig);
       ```

       ```typescript
         });
         ```
