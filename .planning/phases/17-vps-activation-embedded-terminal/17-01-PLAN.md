    **infra/vps/container-manager.ts** (NEW):
    Docker container lifecycle management via dockerode.

    ```typescript
    import Docker from "dockerode";
    const docker = new Docker({ socketPath: "/var/run/docker.sock" });
    ```

    Constants:
    - `const SANDBOX_IMAGE = "openclaw-sandbox-common:bookworm-slim";`
    - `const MEMORY_LIMIT = 512 * 1024 * 1024;` (512MB)
    - `const CPU_LIMIT = 0.5 * 1e9;` (0.5 CPUs in NanoCpus)
    - `const LABEL_KEY = "fleet-factory";`

    Export `async function createAgentContainer(vpsAgentId: string, workspacePath: string, sharedPath: string): Promise<void>`:
    - Check if container with name `vpsAgentId` already exists via `docker.getContainer(vpsAgentId).inspect()` in try/catch
    - If exists and running, stop it first, then remove it
    - If exists and not running, remove it
    - Create new container:
      ```
      docker.createContainer({
        Image: SANDBOX_IMAGE,
        name: vpsAgentId,
        Labels: { [LABEL_KEY]: "true", "tenant": vpsAgentId.split("-").slice(0, -2).join("-") || vpsAgentId },
        Cmd: ["sleep", "infinity"],  // Agent process managed by OpenClaw
        HostConfig: {
          Memory: MEMORY_LIMIT,
          NanoCpus: CPU_LIMIT,
          Binds: [
            `${workspacePath}:/workspace:rw`,
            `${sharedPath}:/shared:ro`,
          ],
          NetworkMode: "bridge",
          RestartPolicy: { Name: "unless-stopped" },
        },
      })
      ```
    - Start the container
    - Console.log the creation

    Export `async function startAgentContainer(vpsAgentId: string): Promise<void>`:
    - `docker.getContainer(vpsAgentId).start()` wrapped in try/catch (ignore "already started" errors)

    Export `async function stopAgentContainer(vpsAgentId: string): Promise<void>`:
    - `docker.getContainer(vpsAgentId).stop()` wrapped in try/catch (ignore "not running" errors)

    Export `async function listTenantContainers(businessSlug: string): Promise<Array<{ id: string; name: string; state: string; vpsAgentId: string }>>`:
    - `docker.listContainers({ all: true, filters: { label: [\`${LABEL_KEY}=true\`] } })`
    - Filter to containers whose Names include the businessSlug
    - Map to simplified objects

    Export `async function stopTenantContainers(businessSlug: string): Promise<number>`:
    - Call `listTenantContainers(businessSlug)`
    - Stop each running container, count stopped
    - Return count

    Export `async function resumeTenantContainers(businessSlug: string): Promise<number>`:
    - Call `listTenantContainers(businessSlug)`
    - Start each non-running container, count started
    - Return count

    Export `async function countRunningAgents(): Promise<number>`:
    - `docker.listContainers({ filters: { status: ["running"], label: [\`${LABEL_KEY}=true\`] } })`
    - Return `.length`

    **infra/vps/openclaw-client.ts** (NEW):
    OpenClaw gateway HTTP and WebSocket client for agent interaction.

    ```typescript
    import { WebSocket } from "ws";
    ```

    Constants:
    - `const GATEWAY_URL = process.env.OPENCLAW_HTTP_URL || "http://127.0.0.1:18789";`
    - `const GATEWAY_WS_URL = process.env.OPENCLAW_WS_URL || "ws://127.0.0.1:18789";`
    - `const AUTH_TOKEN = process.env.OPENCLAW_AUTH_TOKEN || "";`

    Export `async function sendMessageToAgent(sessionKey: string, message: string): Promise<{ response: string; model?: string; tokens?: number }>`:
    - POST to `${GATEWAY_URL}/api/sessions/${sessionKey}/messages`
    - Headers: `Content-Type: application/json`, `Authorization: Bearer ${AUTH_TOKEN}`
    - Body: `{ message }`
    - Parse response JSON
    - If not ok, throw with status
    - Return `{ response: data.content || data.response || "", model: data.model, tokens: data.usage?.total_tokens }`

    Export `async function submitTaskToAgent(sessionKey: string, taskPayload: { taskId: string; title: string; payload: Record<string, unknown> }): Promise<{ success: boolean; result?: Record<string, unknown>; toolsUsed?: string[]; tokenUsage?: { prompt_tokens: number; completion_tokens: number }; error?: string }>`:
    - POST to `${GATEWAY_URL}/api/sessions/${sessionKey}/messages`
    - Body: `{ message: \`Execute task: ${taskPayload.title}\n\nPayload: ${JSON.stringify(taskPayload.payload)}\` }`
    - Parse response, map to task result shape
    - On error, return `{ success: false, error: message }`

    Export `async function checkGatewayHealth(): Promise<{ status: string; sessions: number }>`:
    - GET `${GATEWAY_URL}/api/status` with Bearer auth and 3s AbortController timeout
    - On success, return parsed JSON
    - On failure, return `{ status: "unreachable", sessions: 0 }`

    Export `function streamChatFromAgent(sessionKey: string, message: string, onToken: (token: string) => void, onComplete: (fullResponse: string) => void, onError: (error: string) => void): () => void`:
    - Create WebSocket to `${GATEWAY_WS_URL}/api/sessions/${sessionKey}/stream?token=${AUTH_TOKEN}`
    - On open: send `JSON.stringify({ message })`
    - On message: parse event, if type "token" call `onToken(data.content)`, if type "complete" call `onComplete(data.content)`
    - On error/close: call `onError` if unexpected
    - Return cleanup function that closes the WebSocket
    - Fallback: if WebSocket connection fails, fall back to HTTP POST via `sendMessageToAgent` and call `onComplete` with the full response
