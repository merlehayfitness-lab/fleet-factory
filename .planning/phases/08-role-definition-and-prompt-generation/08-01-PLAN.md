<objective>
Add Role Definition card, Claude-powered prompt generation, and prompt preview/refinement to the agent Config tab.

Purpose: This plan adds the ability for admins to describe agent roles in plain language (description, tone, focus) and have Claude generate structured system prompts with 4 sections (Identity, Instructions, Tools, Constraints). The generation flow is two-step (parse breakdown for confirmation, then generate). Admins can refine prompts via chat-like interaction with diff highlighting, and test prompts with an inline chat. Role definition templates are provided per department type.

Output: 1 SQL migration, 4 core prompt module files, 2 server action files (1 new, 1 modified), 6 UI components (5 new, 1 restructured), updated agent service and exports. Split into 3 tasks: core backend (Task 1), server actions + simpler UI components (Task 2), complex panels + agent config restructure (Task 3).
</objective>

    // Output format: same JSON as GENERATE_META_PROMPT

    // Environment: ANTHROPIC_API_KEY accessed via helper function that throws if missing.
    // Model: "claude-sonnet-4-20250514" for all generation calls.
    // Follow same getApiKey/getClient pattern as packages/core/knowledge/embedder.ts uses for OpenAI.
    //
    // parseRoleDefinition(roleDefinition: RoleDefinition): Promise<ParsedBreakdown>
    //   - Call Claude with PARSE_META_PROMPT as system, role description as user message
    //   - Parse JSON response into ParsedBreakdown
    //   - Return the breakdown for admin confirmation
    //
    // generateSystemPrompt(roleDefinition: RoleDefinition, breakdown: ParsedBreakdown,
    //                       suggestions: ContextSuggestions): Promise<GeneratedPrompt>
    //   - Build user message from confirmed breakdown + role context + checked suggestions
    //   - Call Claude with GENERATE_META_PROMPT as system
    //   - Parse JSON response into 4 sections
    //   - Build fullPrompt by concatenating: "## Identity\n{identity}\n\n## Instructions\n{instructions}\n\n## Tools\n{tools}\n\n## Constraints\n{constraints}"
    //   - Return GeneratedPrompt
    //
    // refineSystemPrompt(currentPrompt: GeneratedPrompt, request: string): Promise<GeneratedPrompt>
    //   - Call Claude with REFINE_META_PROMPT as system
    //   - Include current 4 sections + refinement request as user message
    //   - Parse updated sections, rebuild fullPrompt
    //   - Return updated GeneratedPrompt
    //
    // testPrompt(systemPrompt: string, messages: Array<{role: "user"|"assistant", content: string}>): Promise<string>
    //   - Call Claude with systemPrompt as system message
    //   - Pass messages array as conversation history
    //   - Return assistant response string
    //   - No database persistence -- ephemeral test
    //
    // Error handling: wrap all Claude calls in try/catch.
    // If ANTHROPIC_API_KEY missing, throw descriptive error.
    // If Claude returns invalid JSON, throw parse error with raw response for debugging.
