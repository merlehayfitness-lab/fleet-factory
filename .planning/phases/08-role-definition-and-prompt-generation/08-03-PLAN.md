<objective>
Build the agent setup wizard with knowledge upload, role definition, prompt generation, and sub-agent support.

Purpose: This plan creates the multi-step wizard at /businesses/[id]/agents/new that guides admins through creating a new agent. The wizard flow is: Basic Info (name, department, role) -> Knowledge Upload (agent-specific docs, blocks until processed) -> Role Definition (structured fields with template selector) -> Prompt Generation (Claude generates prompt + SKILL.md with refinement) -> Review (full summary with Create Agent button). It reuses components from 08-01 (RoleDefinitionCard pattern, PromptRefinementPanel, TestChatDialog) and Phase 7 (KnowledgeUploadZone). A curated template library provides starter role definitions per department type.

Output: Role template library, wizard Server Actions, 5 wizard step components, wizard parent component, new route page, agents list "New Agent" button.
</objective>
