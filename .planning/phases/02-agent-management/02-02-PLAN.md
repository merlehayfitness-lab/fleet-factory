    **apps/web/app/(dashboard)/businesses/[id]/agents/[agentId]/page.tsx:**
    A Server Component that:
    - Extracts businessId and agentId from params
    - Creates Supabase server client, calls getUser(), redirects if no user
    - Fetches the agent with department and template data:
      ```
      supabase
        .from('agents')
        .select('*, departments(id, name, type), agent_templates(id, name, system_prompt, tool_profile, model_profile)')
        .eq('id', agentId)
        .eq('business_id', businessId)
        .single()
      ```
    - If agent not found, call notFound()
    - Fetches recent audit_log entries for this agent:
      ```
      supabase
        .from('audit_logs')
        .select('*')
        .eq('business_id', businessId)
        .eq('entity_type', 'agent')
        .eq('entity_id', agentId)
        .order('created_at', { ascending: false })
        .limit(20)
      ```
    - Page heading: agent name with StatusBadge and back link to agents list
    - Render AgentDetailTabs component, passing agent data, template data, audit logs, and businessId
